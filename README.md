# BeyondPhysicsApplication

框架demo下载地址:
http://47.97.113.47:4126/perfectwallpaper_files/apks/beyondPhysics.apk


androidStudio Gradle导入:

先添加jitpack仓库:
allprojects {
    repositories {
        maven { url 'https://www.jitpack.io' }
    }
}
接着导入项目依赖库即可:
dependencies {
    implementation 'com.github.xihuan22d:BeyondPhysics:1.0'
}
